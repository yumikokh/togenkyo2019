# vue-starter

This project is customized for my own usecase. 
Based on [Vue CLI 3](https://cli.vuejs.org/config/)

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm start
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
